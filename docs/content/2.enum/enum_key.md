# Enum Key

## Definition

```
enum class KeyBind {
    ESC,
    A_KEY,
    Z_KEY,
    Q_KEY,
    S_KEY,
    UP_KEY,
    DOWN_KEY,
    LEFT_KEY,
    RIGHT_KEY,
    SPACE,
    ENTER,
    MOUSE_LEFT,
    MOUSE_RIGHT,
    NONE
};
```

## Why

To detect which key is pressed and handle only the keys we want.
